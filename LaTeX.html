1 Basic

1.1 Source

Coded with plain text

1.1.1 Blank

  1. More than one whitespaces is regarded as one
  2. Same as the blank line

1.1.2 Special Characters

Some characters are reserved for special usage 

# $ % ^ & _ { } ~ \

while you can use \ to excape them
we cannot use \\ to excape \ because they are line breaker1

1.1.3 LaTeX commands

Commands are case sensitive

  1. started with \, followed by commands, ended by whitespace, digit or other non-alphabetic characters
  2. one \ and a special character

LaTeX will ignore the whitespace after the command
If you want it, use {} and whitespace after the command
Or just use a special whitespace command

See more commands below

Many commands need parameters enclosed by {} (some by [])

1.1.4 Comments

Follows special character %

1.2 Structure

Each LaTeX document must start with 

\documentclass{...}

then we can add some document control settings, like 

\usepackage{...}

After all the settings are done, start the document with 

\begin{document}

and end the document with 

\end{document}

1.3 A classic process

A simple example source code of LaTeX A complicated example:

  1. Create .tex file (must encoded by ASCII)

  2. use latex to compile .tex file 

    latex foo.tex

  3. preview the .dvi file 

    xdvi foo.dvi

    or transform this to a .ps file 

    dvips -Pcmz foo.dvi -o foo.ps

1.4 Document layout

1.4.1 documentclass

To set the document class: \documentclass[options]{class}
For example: 

\documentclass[11pt, twoside, a4paper]{article}

Classes:

  1. article: for magazines, short reports, technique documents, so on
  2. report: for multi-charpter reports, short books, thesis
  3. book: for books
  4. slides: use it like ppt

Options:

  1. 10pt, 11pt, 12pt: use 10pt by default
  2. a4paper, letterpaper, ...: size of paper
  3. fleqn: Equations will be aligned to the left
  4. leqno: Equations with its number
  5. titlepage, notitlepage: start a new page or not, not by default
  6. onecolumn, twocolumn: one column or two column in one page
  7. twoside, oneside: one side of page or two
  8. openright, openany: new chapter will be presented at single page, or next page available

1.4.2 macro package

To use a macro package: \usepackage[options]{package}
Some packages:

  1. normalfontdoc: organize LaTeX documents (see doc.dtx)
  2. exscale: provide scalable math font (see ltexscale.dtx)
  3. normalfontfontenc: declare font encoding of LaTeX (see ltoutenc.dtx)
  4. ifthen: provide if...then do...otherwise do... cmd (see ifthen.dtx)
  5. latexsym: provide LaTeX symbol in document (see latexsym.dtx)
  6. makeidx: provide cmd of generating index
  7. syntonly: just handle document, no typesetting

  8. inputenc: allow declaration of input encoding like 

    ASCII, ISO Latin-1, ISO Latin-2, 437/850, IBM, Apple Macintosh, so on.

1.5.1 Page Style

Pre-defined format for page header or footer 

\pagestyle{style}

Styles:

  1. plain: no header, footer is page number (at the center) (default)
  2. headings: header is page number and chapter name (at the center), no footer
  3. empty: no header, no footer

use \thispagestyle{style} to change current page style

1.6 Large document

When handling large document, best divide it into a few parts
use \include{filename} to include filename.tex file (opening a new page)

\includeonly{filename,filename,...} to restrict filename

use syntonly package to check syntax error without typesetting 

\usepackage{syntonly}
\syntaxonly

2 Typesetting

2.1 Document structure

Paragraph is for coherence thoughts
Use \\ or blank line well

2.2 Line breaks and pages

2.2.1 Paragraph

  1. \\ or \newline: new line
  2. \\*: break line, no paging
  3. \newpage: new page
  4. \linebreak[n], \nolinebreak[n], \pagebreak[n], \nopagebreak[n]

2.2.2 Cutting words

  1. \hyphenation{word list}: make target word been cut at the given position
  2. \mbox{text}: keep text in one line
  3. \fbox{text}: like \mbox, with a frame

2.2.3 String

Command Example Description
\today 20th October 2015 current date in the current language
\TeX TEX name of your favorite typesetter
\LaTeX LaTeX name of the game
\LaTeXe LaTeX 2e current incarnation of LaTeX

2.2.4 Special characters

  1. Quote: use two single quote to generate a double quote, don't use double quote
  2. Dash: four places to use -:
    1. one -: hyphen
    2. two -: short dash
    3. three -: long dash
    4. in equation: minus
  3. Wave: \~ (small) or \sim (big)
  4. Degree: ^{\circ}
  5. Ellipsis: \ldots
  6. Split two char: \mbox{} to split two char
  7. Phonetic notation and special character
    (Well I don't think I will use these)
